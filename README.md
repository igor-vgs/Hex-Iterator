# HexIteration

The aim of this work is to study generalized iterators and instance nested classes of the Java language.

The project is an implementation of a sequence of integers with an iterator over the maximum digits in their hexadecimal representation.
___________________________________________________________________________________________________________________________________________
Целью данной работы является изучение обобщённых итераторов и экземплярных вложенных классов языка Java.

Представлена реализация последовательности целых чисел с итератором по максимальным цифрам в их 16-ричном представлении.
